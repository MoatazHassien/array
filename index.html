Programming I: Chapter 4 (Arrays) Quiz

1. Which of the following statements correctly declares an array of integers named "numbers" with a size of 10?
A) int numbers = new int[10];
B) int[] numbers = new int(10);
C) int[] numbers = new int[10];
D) numbers int[] = new int[10];
Correct Answer: C
التفسير: وفقاً للصفحة 3 و 25، الصيغة الصحيحة لتعريف المصفوفة هي type[] name = new type[length];.
2. What is the default value of elements in a boolean array immediately after declaration?
A) true
B) false
C) null
D) 0
Correct Answer: B
التفسير: كما هو موضح في الجدول في الصفحة 5، القيمة الافتراضية (Default value) للنوع boolean هي false.
3. Consider the following code. What is the valid range of indexes for the array data?
int[] data = new int[8];
A) 1 to 8
B) 0 to 8
C) 0 to 7
D) 1 to 7
Correct Answer: C
التفسير: الصفحة 2 تذكر أن الـ Index يبدأ من 0. والصفحة 19 توضح أن الـ Legal indexes هي بين 0 و (طول المصفوفة - 1). إذاً من 0 إلى 7.
4. What will be the output of the following code snippet?
int[] list = new int[5]; System.out.println(list[4]);
A) null
B) 0
C) Garbage value
D) Runtime Error
Correct Answer: B
التفسير: عند تعريف مصفوفة من نوع int، يتم تهيئة جميع العناصر بـ 0 تلقائياً (الصفحة 5). العنصر في الـ index 4 هو العنصر الأخير وقيمته 0.
5. Analyze the code below. What is the value of arr[1] after execution?
int[] arr = new int[4]; arr[0] = 2; arr[1] = arr[0] * 3 + 1;
A) 6
B) 0
C) 7
D) 3
Correct Answer: C
التفسير: تم تعيين arr[0] بـ 2. المعادلة هي 2 * 3 + 1 وتساوي 7. هذا تطبيق مباشر لتعديل القيم (الصفحة 6).
6. Which exception is thrown if you try to access numbers[-1]?
A) NullPointerException
B) ArrayIndexOutOfBoundsException
C) InvalidIndexException
D) Compilation Error
Correct Answer: B
التفسير: الصفحة 19 تنص بوضوح على أن استخدام أي index خارج النطاق (مثل القيم السالبة) يسبب ArrayIndexOutOfBoundsException.
7. What is the result of the following operation based on the code below?
int[] a = {10, 20, 30}; int x = a.length; System.out.println(a[x]);
A) 30
B) 10
C) 3
D) ArrayIndexOutOfBoundsException
Correct Answer: D
التفسير: طول المصفوفة (x) هو 3. أقصى index مسموح به هو 2. محاولة الوصول لـ a[3] تسبب خطأ (Exception) كما هو مشروح في الصفحة 19.
8. In a 2D array declared as double[][] matrix = new double[4][5];, what does matrix.length represent?
A) The total number of elements (20)
B) The number of columns (5)
C) The number of rows (4)
D) The size of the first row
Correct Answer: C
التفسير: في المصفوفات ثنائية الأبعاد، name.length تعيد عدد الصفوف (rows). انظر الصفحة 24.
9. What is the output of the following nested indexing code?
int[] nums = new int[5]; nums[0] = 1; nums[1] = 3; nums[3] = 0; System.out.println(nums[nums[1]]);
A) 1
B) 3
C) 0
D) ArrayIndexOutOfBoundsException
Correct Answer: C
التفسير: لنحللها خطوة بخطوة. nums[1] قيمتها 3. إذاً نحن نطلب nums[3]. وقيمة nums[3] تم تعيينها لتكون 0. (مبدأ استخدام عنصر كـ Index مشروح صفحة 8).
10. How do you find the number of columns in the first row of a 2D array named grid?
A) grid.length
B) grid[0].length
C) grid.size()
D) grid[1].size
Correct Answer: B
التفسير: الصفحة 24 توضح أن name[0].length تعيد عدد الأعمدة في الصف الأول.
11. Which loop structure is typically used to read array elements from the user?
A) for (int i=0; i <= length; i++)
B) for (int i=1; i < length; i++)
C) for (int i=0; i < length; i++)
D) for (int i=1; i <= length; i++)
Correct Answer: C
التفسير: المصفوفات تبدأ من 0 وتنتهي عند أقل من length بواحد. الخيار C هو الصحيح لتجنب الخطأ. (الصفحات 10-12).
12. Based on the "Quick array initialization" logic, what is the size of the following array?
int[] vals = {5, 10, 15, 20, 25};
A) 4
B) 5
C) It must be specified manually.
D) 6
Correct Answer: B
التفسير: الصفحة 17 تذكر أن المترجم (compiler) يقوم بحساب الحجم تلقائياً بناءً على عدد القيم المدخلة، وهي هنا 5 قيم.
13. In the statement int[] data = new int[x % 5 + 2];, if x is 12, what is the length of the array?
A) 2
B) 4
C) 5
D) 12
Correct Answer: B
التفسير: حسب الصفحة 5، يمكن للحجم أن يكون تعبيراً رياضياً. 12 % 5 يساوي 2. ثم 2 + 2 يساوي 4.
14. What is the output of the following code snippet?
int[] arr = new int[5]; for(int i=0; i < 5; i++) { arr[i] = i * 2; } System.out.println(arr[3]);
A) 3
B) 5
C) 6
D) 9
Correct Answer: C
التفسير: الحلقة تملأ المصفوفة. عندما تكون i=3، فإن arr[3] = 3 * 2، مما ينتج 6. (مشابه للمثال في الصفحة 10).
15. Which of the following correctly defines a 2D array using quick initialization?
A) int[][] m = { {1,2}, {3,4} };
B) int[][] m = new int[2][2] {1,2,3,4};
C) int m[][] = { (1,2), (3,4) };
D) int[][] m = [1,2][3,4];
Correct Answer: A
التفسير: الصيغة الصحيحة للتهيئة السريعة للمصفوفات ثنائية الأبعاد تستخدم الأقواس المتعرجة المتداخلة كما في الصفحة 22.
16. If double[][] temps = new double[3][5];, what does temps[2] refer to?
A) The element at row 2, column 0.
B) The entire third row.
C) The entire second column.
D) A single double value.
Correct Answer: B
التفسير: وفقاً للصفحة 23، temps[2] تشير إلى الصف الثالث (third row) بأكمله كمصفوفة.
17. Look at the code block. What happens if the user enters 5 for the size?
Scanner input = new Scanner(System.in); int size = input.nextInt(); int[] arr = new int[size]; System.out.println(arr[size]);
A) Prints 0
B) Prints 5
C) Throws ArrayIndexOutOfBoundsException
D) Prints the memory address
Correct Answer: C
التفسير: إذا كان الحجم 5، فإن الـ indexes الصحيحة هي 0، 1، 2، 3، 4. محاولة طباعة arr[5] تقع خارج النطاق. (صفحة 19).
18. To calculate the sum of elements in an array salaries, what should replace the `???`?
double sum = 0; for (int i=0; i < salaries.length; i++) { ??? }
A) salaries[i] = sum;
B) sum = salaries[i];
C) sum += salaries[i];
D) sum = sum + i;
Correct Answer: C
التفسير: لحساب المجموع التراكمي، يجب إضافة قيمة العنصر الحالي إلى المجموع السابق sum = sum + salaries[i]. (انظر كود الصفحة 13).
19. What data type is the "index" of an array always considered to be?
A) double
B) integer (int)
C) boolean
D) String
Correct Answer: B
التفسير: الصفحة 2 تعرف الـ index بأنه "0-based integer". لا يمكن استخدام الفواصل العشرية كفهرس.
20. What is the value of count after this loop?
int[] nums = {-10, 20, -30, 40}; int count = 0; for(int i=0; i < nums.length; i++) { if(nums[i] > 0) count++; }
A) 1
B) 2
C) 3
D) 4
Correct Answer: B
التفسير: الكود يعد الأرقام الموجبة فقط. الأرقام الموجبة في المصفوفة هي 20 و 40. إذاً العدد هو 2. (منطق مشابه لمثال الصفحة 15).
21. In Java, an array is best described as:
A) A primitive data type.
B) An object that stores many values of the same type.
C) A function to calculate math.
D) An infinite storage container.
Correct Answer: B
التفسير: التعريف الحرفي الموجود في الصفحة 2: "Array: object that stores many values of the same type".
22. Given int[][] nums = new int[3][4];, which loop correctly iterates over all rows?
A) for(int i=0; i < nums[0].length; i++)
B) for(int i=0; i < nums.length; i++)
C) for(int i=0; i <= 3; i++)
D) for(int i=1; i < 3; i++)
Correct Answer: B
التفسير: للمرور على الصفوف، نستخدم الحلقة الخارجية التي تتوقف عند nums.length (عدد الصفوف). الخيار B هو الصحيح وفقاً للصفحة 25.
23. Can the size of an array be changed after it is created (e.g., new int[5])?
A) Yes, by assigning a new value to .length.
B) Yes, automatically when adding elements.
C) No, the size is fixed upon creation.
D) Yes, if it is a double array.
Correct Answer: C
التفسير: مفهوم المصفوفات في هذا الفصل (والجافا عموماً) أن الحجم يحدد عند الإنشاء (new int[n]). لإدخال عناصر أكثر، لا يمكن توسيعها بل يجب إنشاء واحدة جديدة (لم يذكر الملف التوسيع، مما يعني أن الحجم ثابت).
24. Identify the logic error in this loop designed to print array elements:
int[] arr = {1, 2, 3}; for (int i = 0; i <= arr.length; i++) { System.out.println(arr[i]); }
A) Syntax error in print statement.
B) The loop starts at 0.
C) The condition i <= arr.length causes an OutOfBounds exception at the end.
D) No error, it runs perfectly.
Correct Answer: C
التفسير: الخطأ الشائع المذكور ضمنياً في الأمثلة (الصفحة 10 و 19). الشرط يجب أن يكون i < length. وجود علامة = يعني أن الحلقة ستحاول الوصول للـ index المساوي للطول، وهو غير موجود.
25. What is the correct syntax to access the element at Row 0, Column 3 in a 2D array named map?
A) map[0, 3]
B) map[3][0]
C) map[0][3]
D) map(0)(3)
Correct Answer: C
التفسير: الصفحة 20 و 21 توضح أن الوصول للعناصر في مصفوفة ثنائية الأبعاد يتم عبر [row][col].
26. Which of the following is NOT a valid array declaration?
A) int[] a = new int[5];
B) double[] b = new double[x * y];
C) int c[] = new int[5];
D) int[] d = new int[];
Correct Answer: D
التفسير: عند استخدام كلمة new، يجب تحديد حجم المصفوفة بين القوسين []. الخيار D يفتقد للحجم وهذا خطأ (syntax error). (راجع صفحة 25).
27. What happens if you run: int[] arr = new int[5]; arr[5] = 10;?
A) The array size increases to 6.
B) The value 10 is ignored.
C) An exception is thrown because index 5 is out of bounds.
D) It sets the last element to 10.
Correct Answer: C
التفسير: المصفوفة حجمها 5، الـ indexes هي 0,1,2,3,4. الـ index 5 غير موجود. (صفحة 19).
28. In the example Scanner input = new Scanner(System.in); int n = input.nextInt(); int[] numbers = new int[n];, where does the array length come from?
A) It is fixed to 10.

29. What is the exact output of this snippet?
int[] nums = {1, 2, 3}; System.out.println(nums);
A) 1 2 3
B) {1, 2, 3}
C) A memory address string (hash code).

التفسير: على الرغم من أن الملف لا يذكر هذا صراحة كنص، إلا أن جميع الأمثلة في الملف تستخدم حلقات (loops) لطباعة العناصر (مثل صفحة 10 و 12). هذا يعني ضمناً أن طباعة اسم المصفوفة مباشرة لا يطبع محتوياتها بل عنوانها. (هذا سؤال استنتاجي صعب).
30. "Element" refers to:

التفسير: تعريف مباشر من الصفحة 2: "Element: One value in an array".
